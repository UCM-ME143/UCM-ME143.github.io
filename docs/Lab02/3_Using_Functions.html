Using Functions

It can get tiresome to keep having to copy-paste or rerun the rotation matrix equations every time. It's easier if we rewrite it as a programming function.

Writing a Function

To write a function, click on the arrow below NEW in Matlab, then select either 'Function' or 'Live Function' Use 'Function' for regular *.m files and use 'Live Function' for *.mlx files.
In the new file, we can write a function. To test, let's do a simple example. Use the example code below:
function out = function_name(function_input1,function_input2)
out = function_input1 + function_input2;
end
Save the file as function_name.mlx in the same workspace folder. Now we can use it - once the file is saved, you can call it within the Command Window or within a Live Script. Call it like you would for any other function:
testing = function_name(1,2);
 
disp(testing);
3
Thats how simple it is to make a function - now this isn't a usefull function, but you could imagine if you had to repeat a complicated series of equations how valuable this would be.

Rotation Functions

Lets make a function to generate a rotation matrix for Phi. Create a new Live Function and copy the code into the file.
function R = rotate_phi(angle)
angle = deg2rad(angle);
R = [ 1 , 0, 0;
0 , cos(angle), sin(angle);
0 ,-sin(angle), cos(angle)];
end
Now we can do
Rx_phi = rotate_phi(50);
disp(Rx_phi);
1.0000 0 0 0 0.6428 0.7660 0 -0.7660 0.6428
Now we can do our rotation around the x axis without having to copy/paste the code.
Go ahead and do the same for the other rotations. Once that's ready, you can make the following calls:
Rz_psi = rotate_psi(10);
Ry_the = rotate_theta(-30);
And now we can calculate our Transform To BODY from EARTH
T_2BFE = Rx_phi*Ry_the*Rz_psi;
disp(T_2BFE);
0.8529 0.1504 0.5000 -0.4888 0.5665 0.6634 -0.1835 -0.8102 0.5567
And if we want to go the other way, we take the transpose. This would give us the Transform To EARTH from BODY
T_2EFB = T_2BFE';
disp(T_2EFB);
0.8529 -0.4888 -0.1835 0.1504 0.5665 -0.8102 0.5000 0.6634 0.5567
